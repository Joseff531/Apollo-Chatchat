## Retrieval-Augmented Generation (RAG)
- Vector databases (FAISS, Milvus, Chroma, pgvector, Elasticsearch)
- Chunking strategies (recursive, semantic, sentence-window)
- Embedding models (e.g. bge-large, e5, OpenAI text-embedding-3)
- Re-ranking with cross-encoders or LLM-as-judge

## Agent patterns
- ReAct-style reasoning + tool calls
- Function calling / OpenAI tools API
- Multi-step planning (plan-and-execute, tree-of-thought)

## Prompting techniques
- Few-shot examples and chain-of-thought
- System / role messages and persona steering
- Structured output via JSON schema
