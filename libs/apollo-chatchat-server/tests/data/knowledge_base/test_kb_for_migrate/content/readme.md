#### Apollo-Chatchat

Apollo-Chatchat is an open-source, offline-deployable RAG and Agent application built on top of large language models such as ChatGLM and Qwen, using the LangChain framework.

#### Features

- Local knowledge-base Q&A
- Pluggable inference backends (Xinference, Ollama, LocalAI, FastChat)
- Agent / tool calling
- File-based RAG with BM25 + KNN retrieval
